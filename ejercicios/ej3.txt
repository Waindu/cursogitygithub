B) Actividad Grupal - (grupos de 4)

crear repositorio github

los demas, clonar el repositorio una vez se haya subido
agregar colaboradores
crear un gitignore para ignorar todo tipo de archivos .jpg menos uno que deberá subirse al origin o repositorio remoto
crear una historia en master
crear rama develop y una pequeña historia en esta rama, hacer un merge con master
crear una rama para cada persona y su historia (hacer commits de estas ramas y merges a develop)
crear conflictos de merge entre ramas y solucionarlos en grupo
hacer pull request para mergear de las ramas propias a develop
hacer cambios y commits directamente desde github
mirar historias y blame de archivos en github
